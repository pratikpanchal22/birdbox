1. Create file birdSongBoxHwController.service with following content:

[Unit]
Description=BirdBox Hardware Controller Service
After=multi-user.target

[Service]
Type=idle
WorkingDirectory=/home/<appUser>/birdbox
ExecStart=/usr/bin/python3 /home/<appUser>/birdbox/hardwareController.py
Restart=always
User=<appUser>
Group=<appUserGroup>

[Install]
WantedBy=multi-user.target

2. Move that file to /etc/systemd/system

3. Make it readable/writable/execuatble by root:
sudo chmod u+rwx /etc/systemd/system/birdSongBoxHwController.service 

4. Enable service to auto-start at boot:
sudo systemctl enable birdSongBoxHwController

5. To start, stop and check status manually:
sudo systemctl <start>, <stop>, <status> birdSongBoxHwController
